01

SMART HOME IoT NETWORK

Open

Close

Type:Personal project

This project focuses on the design and implementation of an Internet of Things (IoT) network for a smart home environment. I integrated sensors and devices such as temperature monitors, motion detectors, and smart lighting, All connected through a local network managed by a Raspberry Pi server. The system was programmed to collect and visualize real-time data, improving energy efficiency and comfort.

Technologies:Raspberry Pi, Node-RED, MQTT, Wi-Fi, JSON
